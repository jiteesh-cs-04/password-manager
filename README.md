# Password Manager
A desktop Password Manager built with Python and Tkinter.
## Features
- Add websites
- Delete websites
- Add accounts
- Delete accounts
- Change passwords
- Generate strong passwords

- Object-Oriented Programming (OOP)
- Tkinter GUI development
- JSON file handling
- Git and GitHub
- Organizing Python projects into modules
